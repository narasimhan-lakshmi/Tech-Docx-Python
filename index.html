Introduction

Python is a versatile and widely-used programming language known for its readability and simplicity. It is a high-level, interpreted language, which means you don't need to worry about low-level details such as memory management. Python's syntax is designed to be clear and concise, making it an excellent choice for beginners and experienced developers alike.

Variables

In Python, a variable is a name that refers to a value. It is a way to store information that can be referenced and manipulated in a program.

Variable names must start with a letter (a-z, A-Z) or an underscore (_). The subsequent characters can be letters, numbers, or underscores. Variable names are case-sensitive (e.g., myVar and myvar are different). Avoid using reserved words as variable names (e.g., if, while, class, etc.).

# Variable assignment my_variable = 42
Variable Scope

The scope of a variable defines where it can be accessed. Variables defined inside a function have local scope, while variables defined outside functions have global scope.

global_variable = 10 # Global scope def my_function(): local_variable = 20 # Local scope
Global Variable

In Python, a global variable is a variable declared outside of any function, and it can be accessed and modified from any part of the code, including both inside and outside functions. Global variables are defined at the top level of the script or module, and they maintain their value throughout the entire program.

# Global variable global_variable = 10 def my_function(): # Accessing the global variable inside a function print("Inside the function:", global_variable) # Calling the function my_function() # Modifying the global variable outside the function global_variable = 20 # Accessing the modified global variable print("Outside the function:", global_variable)
Constants

In Python, there is no strict concept of constants like in some other programming languages (e.g., C or Java), where you can explicitly declare a variable as a constant, and its value cannot be changed once set.

However, it's a common convention in Python to use uppercase names for variables whose values are not intended to be changed during the execution of a program, treating them as if they were constants.

If Else statement

The if-else statement in Python allows you to execute different blocks of code based on whether a specified condition evaluates to True or False.

The basic syntax is as follows:

if condition: # Code to execute if the condition is True else: # Code to execute if the condition is False

Example:

x = 10 if x > 5: print("x is greater than 5") else: print("x is not greater than 5")

While statement

The while statement in Python is used to create a loop that repeatedly executes a block of code as long as a specified condition is True.

The general syntax of a while loop is as follows:

while condition: # Code to execute while the condition is True

Example:

count = 1 while count <= 5: print(count) count += 1

It's crucial to ensure that the loop will eventually exit to prevent an infinite loop. You should include a mechanism inside the loop that will eventually make the condition False.

Function Declaration

In Python, you can declare a function using the def keyword. The basic syntax for a function declaration is as follows:

def function_name(parameters): # Code to execute return result # Optional return statement

Here's a simple example of a function that adds two numbers:

def add_numbers(x, y): sum_result = x + y return sum_result
